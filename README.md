# 旅行先提案アプリ

## 概要

旅行先提案アプリは、ユーザーの希望条件に合わせて旅行先候補を提案し、選んだ旅行先の詳細な旅行プランを生成する Streamlit アプリです。

現在地、同行者、旅行期間、予算、気分、国内・海外の希望、自由記述の要望を入力すると、OpenAI API を使って旅行先候補を3件提案します。提案された旅行先については、現地の天気予報を確認したり、AIと相談しながら詳細プランを調整することも可能です。

## 主な機能

- 条件入力による旅行先候補の生成
- 3件の旅行先提案
- 提案先の現地天気予報の表示
- 選択した旅行先の詳細プラン生成
- AIとの追加相談によるプラン調整
- 旅行プランのテキストダウンロード

## 使用技術

- Python
- Streamlit
- OpenAI API
- Open-Meteo API
- 国土地理院住所検索 API
- python-dotenv
- requests

## セットアップ

### 1. リポジトリを取得

```bash
git clone <repository-url>
cd travel_rec
```

すでにこのリポジトリを手元に用意している場合は、この手順は不要です。

### 2. 仮想環境を作成

Windows PowerShell の例:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

macOS / Linux の例:

```bash
python -m venv .venv
source .venv/bin/activate
```

### 3. 依存関係をインストール

```bash
pip install -r requirements.txt
```

## 環境変数

プロジェクト直下に `.env` ファイルを作成し、OpenAI APIキーを設定します。

```env
OPENAI_API_KEY=your_openai_api_key_here
```

`.env` は `.gitignore` に含まれているため、APIキーをリポジトリへ誤って含めないようになっています。

## 起動方法

仮想環境を有効化した状態で、以下のコマンドを実行します。

```bash
streamlit run travelapp.py
```

起動後、ブラウザで Streamlit アプリが開きます。自動で開かない場合は、ターミナルに表示されたローカルURLへアクセスしてください。

## 使い方

1. サイドバーで居住地、同行者、旅行開始日、旅行期間、予算、気分、旅行先タイプを入力します。
2. 必要に応じて、自由記述欄に具体的な希望を入力します。
3. 「旅行先を提案」ボタンを押すと、AIが旅行先候補を3件生成します。
4. 各候補の天気予報を確認できます。
5. 気になる旅行先の詳細プランボタンを押すと、具体的な旅行プランが生成されます。
6. チャット欄からAIに追加の質問や修正依頼ができます。
7. 完成した旅行プランはテキストファイルとしてダウンロードできます。

## 注意事項

- OpenAI API を利用するため、実行には有効な `OPENAI_API_KEY` が必要です。
- OpenAI API の利用量に応じて料金が発生する場合があります。
- 天気情報は Open-Meteo API から取得しています。
- 地名から緯度・経度を取得するために、国土地理院住所検索 API を利用しています。
- 外部APIのレスポンス状況や入力された地名によっては、天気情報を取得できない場合があります。
- 生成される旅行先やプランはAIによる提案です。実際の営業日、料金、交通手段、予約可否などは事前に公式情報を確認してください。
